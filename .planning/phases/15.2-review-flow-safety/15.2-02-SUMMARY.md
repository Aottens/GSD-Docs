## Decisions Made

- Used prop-sync useEffect pattern: only syncs activePhaseNumber into selectedPhaseNumber when selectedPhaseNumber is still undefined. Once engineer selects manually, prop polling changes are ignored.
- ReviewProvider phaseNumber now derives from selectedPhaseNumber (user intent), not activePhaseNumber (polling signal). This is the key stability fix for QUAL-06.
- Layout restructured to flex-col: outer div is flex-col, toolbar is shrink-0, existing split panel container changes from h-full to flex-1. This preserves all drag-resize behavior.
