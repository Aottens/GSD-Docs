**Step 1: Add `loadConversation()` to `useDiscussionStream.ts`**

Add a new function `loadConversation` that:
1. Sets the `conversationId` state
2. Fetches all messages from the messages API endpoint
3. Filters out system messages (role !== 'system')
4. Sets the messages state with historical messages
5. Extracts decisions from message metadata (look for messages with `message_type === 'check_in'` that have `metadata_json.all_decisions`) and populates the decisions state
6. Determines if conversation is active or completed and sets viewMode accordingly
7. Sets `currentTopic` from the last `topic_boundary` message with status "starting"

```typescript
const loadConversation = useCallback(async (projectId: string, convId: number) => {
  try {
    setError(null)
    projectIdRef.current = projectId
    setConversationId(convId)

    // Fetch conversation details for status
    const conversation = await api.get<{ id: number; status: string; summary_data: any }>(
      `/projects/${projectId}/discussions/${convId}`
    )

    // Fetch all messages
    const msgs = await api.get<Message[]>(
      `/projects/${projectId}/discussions/${convId}/messages?limit=500`
    )

    // Filter out system messages
    const visibleMessages = msgs.filter((m: Message) => m.role !== 'system')
    setMessages(visibleMessages)

    // Extract decisions from summary_data or check-in messages
    const allDecisions: Decision[] = []
    // First try summary_data (persisted state)
    if (conversation.summary_data?.decisions) {
      allDecisions.push(...conversation.summary_data.decisions)
    }
    setDecisions(allDecisions)

    // Find current topic from last topic_boundary
    const lastBoundary = [...visibleMessages]
      .reverse()
      .find(m => m.message_type === 'topic_boundary' && m.metadata_json?.topic_boundary?.status === 'starting')
    if (lastBoundary?.metadata_json?.topic_boundary) {
      setCurrentTopic(lastBoundary.metadata_json.topic_boundary.topic)
    }

    // Check for completion
    const hasCompletion = visibleMessages.some(m => m.message_type === 'completion_card')
    if (hasCompletion || conversation.status === 'completed') {
      setCompletionData({
        message: 'Bespreking afgerond',
        decisions_count: allDecisions.length,
        topics_covered: [],
      })
    }
  } catch (err) {
    setError(err instanceof Error ? err.message : 'Failed to load conversation')
  }
}, [])
```

Add `loadConversation` to the return object of `useDiscussionStream()`. Add it to the `UseDiscussionStreamReturn` interface too.

} else if (event.event === 'check_in') {
  // Add check-in message to chat (with decision summary)

    id: Date.now(),

    role: 'assistant',
