       c. **Theme store** (`src/stores/themeStore.ts`): Zustand store with persist middleware:
          - State: `theme: 'light' | 'dark' | 'system'`
          - Action: `setTheme(theme)` — applies class to `<html>`, respects system preference for 'system' mode
          - Listen for system preference changes via matchMedia

       d. **Theme toggle** (`src/components/layout/ThemeToggle.tsx`): Dropdown menu with Sun/Moon/Monitor icons for light/dark/system options. Use shadcn dropdown-menu component.

       - Handle errors consistently: throw structured errors with status code and message
       - Do NOT use axios (keep bundle light) — use native fetch with proper typing

    8. **Query client** (`src/lib/queryClient.ts`):
       - Configure QueryClient with staleTime: 5 minutes, retry: 1
       - Export for use in App.tsx provider

        - Include ReactQueryDevtools in development
